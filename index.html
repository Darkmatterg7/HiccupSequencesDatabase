About Hiccup Sequences

A (j, x, y, z)-hiccup sequence is a strictly increasing sequence of positive integers a(n) introduced in 2003 by Benoit Cloitre and formally solved in 2026 by Fokkink and Joshi.

Formal Definition

a(1) = x a(n) = { a(n−1) + y,    if (n − j) ∈ { a(k) : k < n }
a(n−1) + z,    otherwise for n ≥ 2, where j, x ≥ 0 and y, z ≥ 1 (y ≠ z)

Lookback Offset (j): Defines how many positions to look back. For j = 0, we check if the current index n is present in the sequence generated so far. For j = 1, we check if the previous term n - 1 was present.

Worked Example: A086398 (j=1, x=1, y=4, z=2)

a(1) = 1
a(2): (2-1) = 1 ∈ {1} → YES. Add y (+4) → a(2) = 1+4 = 5
a(3): (3-1) = 2 ∈ {1,5} → NO. Add z (+2) → a(3) = 5+2 = 7
a(4): (4-1) = 3 ∈ {1,5,7} → NO. Add z (+2) → a(4) = 7+2 = 9
a(5): (5-1) = 4 ∈ {1,5,7,9} → NO. Add z (+2) → a(5) = 9+2 = 11
a(6): (6-1) = 5 ∈ {1,5,7,9,11} → YES. Add y (+4) → a(6) = 11+4 = 15
Sequence: 1, 5, 7, 9, 11, 15, 17, 21, 23, 27, 29, 33, ...

Mathematical Significance: When the gap values satisfy |y - z| = 1, the sequences can often be written as Beatty Sequences of the form ⌊n·α + β⌋ for some irrational slope α. For sequences where |y - z| > 1, they are generated by substitutions/morphisms, reflecting highly structured fractals that are automatic or morphic.

Citations: Fokkink and Joshi (2026) solved the self-referential sequences mathematically, proving long-standing conjectures (such as those by Clark Kimberling) and mapping translation lemmas (Lemma 9) that shift index behaviors dynamically.
